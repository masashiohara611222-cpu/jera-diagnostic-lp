# METAカルーセル 推奨クリエイティブ

元のカルーセルを「共感型／総合訴求」の推奨案として残し、感情トリガー別の4案と比較します。

## 共感型／総合訴求（推奨・基準案）

- 画像: `../meta-carousel-01.png`〜`../meta-carousel-05.png`
- フック: 「日本での次のキャリア、なんとなく迷っていませんか？」
- 狙い: 強く煽らず、まだ転職を決めていない潜在層の漠然とした迷いに寄り添う
- 推奨本文: 日本での次のキャリアを、まだ決めていなくても大丈夫。5つの質問で、あなたに合う働き方のヒントを1分チェック。
- 特徴: 診断内容、結果、相談導線まで一連の体験が伝わり、幅広い層に配信しやすい
- 推奨運用: 初回配信の基準クリエイティブとして使用する

## 不安刺激型

- 画像: `meta-carousel-01-anxiety.png`
- フック: 「今の給与、実は損してるかも？」
- 狙い: 現状維持への不安を顕在化
- 推奨本文: 今の経験やスキルに、給与や働き方は合っていますか？ 5つの質問で、あなたのキャリアタイプを1分チェック。
- 注意: 強い不安訴求のため、ネガティブコメント率も確認する

## 好奇心型

- 画像: `meta-carousel-01-curiosity.png`
- フック: 「日本の介護で、給与アップできる？」
- 狙い: 転職意向が低い人にも、答えを知りたい気持ちで開いてもらう
- 推奨本文: 日本の介護・病院で働くあなたへ。経験や日本語、希望条件から、次のキャリアのヒントを診断します。
- 注意: 幅広く配信しやすい基準クリエイティブとして使用する

## 承認欲求型

- 画像: `meta-carousel-01-approval.png`
- フック: 「あなたのスキル、日本でどこまで通用する？」
- 狙い: 経験を認めてほしい気持ちと自己効力感に働きかける
- 推奨本文: 現場経験、日本語、資格は大切な強みです。あなたの良いところと、条件アップの可能性を整理してみませんか？
- 注意: 経験年数が長い層や資格保有者の反応を分けて確認する

## 未来想像型

- 画像: `meta-carousel-01-future.png`
- フック: 「3年後、どんな働き方をしていたい？」
- 狙い: 給与だけでなく、安定や生活支援を含む将来像を想像してもらう
- 推奨本文: 給与を上げたい。安心して長く働きたい。生活や手続きも相談したい。あなたが大切にしたい未来から、次の一歩を考えます。
- 注意: 即時転職より情報収集層の保存・診断開始率を重視する

## 比較する指標

1. 1枚目から2枚目への閲覧率
2. 診断開始率
3. 診断完了率
4. Messengerクリック率
5. ネガティブコメント・非表示率

CTRだけで勝敗を決めず、Messengerクリック単価まで比較します。

## 推奨する配信順

1. 共感型／総合訴求を基準案として配信
2. 好奇心型で診断開始率を比較
3. 承認欲求型で経験者・資格保有者への反応を比較
4. 未来想像型で情報収集層の反応を比較
5. 不安刺激型は少額から始め、反発率も含めて評価
